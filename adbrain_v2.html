PERFIL DO NEGÓCIO
📊 Visão Geral
Meta Access Token
Token de acesso à Meta API. Gere em developers.facebook.com/tools/explorer com permissões ads_read e ads_management.
⚡ Dura 1-2h. Para uso contínuo, gere um token de longa duração (60 dias).
ROAS
Receita ÷ Gasto. ROAS 2.5x com comissão de $200 significa $500 de receita para cada $200 gasto. Abaixo de 2.5x = no limite da lucratividade.
CPA
Custo por aquisição (venda). Com comissão de $200: CPA até $80 = lucrativo. CPA acima de $120 = pausar. Entre $80-120 = monitorar.
Conversões vs BuyGoods
A Meta usa janela de atribuição de 7d clique + 1d view. O BuyGoods mostra vendas reais. Sempre tome decisões pelo CPA real do BuyGoods.
⚠ Discrepância grande pode indicar compras canceladas ou problema de atribuição.
🏗️ Estrutura da Campanha
ABO vs CBO
ABO — budget no ad set, mais controle. CBO — Meta distribui o budget automaticamente. Para testes: ABO. Para escalar criativos validados: CBO.
Bid Cap
Limite de lance por leilão. Use 1.5x–2x o CPA alvo. Muito baixo = campanha não entrega. Muito alto = CPA estoura.
⚠ Bid Cap muito conservador pode paralisar a entrega.
Fase de Aprendizado
Em aprendizado — menos de 50 eventos/semana, evitar mudanças. Limitado — consolidar ad sets. Ativo — pode otimizar normalmente.
Compras no Pixel (7d)
Meta precisa de 50+ compras/semana para otimizar bem. Abaixo disso, considerar mudar o evento de otimização para Iniciar Checkout ou Adicionar ao Carrinho.
💰 BuyGoods API
API Key BuyGoods
Solicite ao seu gerente de afiliados (Eduard ou Dane). Acessa os endpoints: sales/by-day, sales/by-hour, sales/by-subid, commissions.
SubID
Parâmetro no link de afiliado que identifica o criativo. Configure como o nome do criativo (ex: PT5745-H9) para cruzar com o Meta Ads automaticamente.
⚡ Formato: seulink.com?sub=PT5745-H9
